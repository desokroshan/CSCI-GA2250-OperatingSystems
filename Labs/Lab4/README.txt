Driver Script:
driver_script contains invocations for the test cases on website.

sample_invocation : 
java Main 10 10 20 1 10 lru 0

Structure:
Main.java reads the input and generates references for each processes.

Pager.java handles the references generated by Main.java

Process.java models processes and the pages

Assumption:
The scripts assumes presence of a file names random.txt containing random numbers in the current directory.
To use a different filename or path, change the same at line 35 in Main.java
